# Flask ToDo App

A simple task management system built with Python, Flask, and SQLite.

## Features

- User registration and login
- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks by category and status
- Task analysis with charts
- Generate PDF report of tasks

## Installation

1.  Clone the repository:


```
   git clone https://github.com/Kacper777Js/Flask-ToDo-app.git
   cd Flask-ToDo-app
```

2. Install dependencies:
```bash

pip install -r requirements.txt
```

3. Run the application:

```bash

python app.py

```
4. Open your web browser and go to
``` 
http://127.0.0.1:5000/

```

## Usage

- Register a new account or log in with existing credentials.
- Use the main page to add tasks with title, description, priority, and category.
- Mark tasks as done, delete them, or filter by category/status.
- Visit the "Analysis" page for charts and PDF reports.

## Technologies Used
